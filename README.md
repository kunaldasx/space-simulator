# Known Space Explorer 🚀

A cinematic space exploration web app built with Streamlit.

## Journey Flow
1. **Launch** — Hit the launch button and watch your shuttle lift off from Earth
2. **Space** — Float in orbit with Earth behind you, then choose to explore
3. **Solar System** — All 8 planets orbiting the Sun; tap any planet for details
4. **Observable Universe** — Zoom out to see the full 93-billion-light-year cosmos

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Community Cloud (Free Public URL)

1. Push this folder to a **GitHub repository**
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Sign in with GitHub
4. Click **New app**
5. Select your repo, branch (`main`), and set main file path to `app.py`
6. Click **Deploy** — your public URL will be ready in ~2 minutes

Your app will be live at:
`[https://<your-app-name>.streamlit.app](https://spacesimulator.streamlit.app/)`
